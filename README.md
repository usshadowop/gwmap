# gwmap

An interactive map of local hobby stores that offer discounts on Games Workshop products. Built with [Leaflet](https://leafletjs.com/) and OpenStreetMap, hosted on GitHub Pages.

## Running locally

Serve the directory with any static file server, e.g.:

```sh
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Adding a store

See [CONTRIBUTING.md](CONTRIBUTING.md).

## Deploying

Enable GitHub Pages in the repo settings: **Settings > Pages > Source**, select the branch to deploy (e.g. `main`) and the root folder. No build step is required.
